(1)改訂歴
 - 2014年 4月29日: 初版

(2)はじめに
このリポジトリは、SUZUKI PLAN - Video Game System 2(GameDaddy)のSDK（兼ソースコード）です。
 - 対応OS（開発環境）= Windowsのみ
 - 対応OS（動作環境）= Windows, Android(2.3.3以降), iOS(4.3以降)

(3)使用許諾
 - 本ソフトウェアは、フリーソフトウェアとして配布します
 - 本ソフトウェアを作成したプログラム（VGS製アプリ）の著作権は開発者に帰属します
 - VGS製アプリの商用 or 非商用については不問です
 - VGS製アプリを商用配布される場合でも、SUZUKI PLANへのライセンス料等の支払いは不要です
 - 本ソフトウェア(GameDaddy)を改変した亜流品（亜流SDK）を作成して頂いて構いません
 - 亜流SDKを配布する場合、公開ドキュメントでその旨を明記してください
 - VGS製アプリ、本ソフトウェア及び亜流SDKを利用して生じた不利益につき、SUZUKI PLANは補償義務を負いません

(4)使い方 - 環境設定
 1. git cloneでリポジトリを複製する（※以下、「C:\VGS2」に複製した想定で説明）
 2. 環境変数の設定
  - PATH: ;C:\VGS2\BIN を追記する
  - VGS2_HOME: C:\VGS2 を設定する
 3. 概ね、初代VGSと同じ要領(vgs2mkpj)でプロジェクトを作成します

[補足事項]
 - iOSへ簡単にポーティングできますが、プロジェクト上で作れるのは、WindowsとAndroidのアプリのみです。
 - 初代VGSと同様の前提ソフトウェアが必要です
 - 現状、初代VGSを使い熟せている方でなければ使えないと思います
 - マニュアルはまだありません（将来的には作ってGitHubで共有予定です）
 - Android版は初期状態でAdMobによる広告表示に対応しています
 - プロジェクトを作成後、javaソースの "ca-app-pub-adcode" という部分をadコードに置換してください
